# Hospital-Appointment-Management-System
ASP.NET Web Forms based hospital appointment management system using C#, SQL Server, and ADO.NET

# Technologies Used
- ASP.NET Web Forms
- C#
- SQL Server
- ADO.NET
- JavaScript

# Features
- Patient Registration
- Appointment Status Update (Pending to Confirmed)
- Record Deletion
- Form Validation (Client & Server Side)
- Parameterized SQL Queries
